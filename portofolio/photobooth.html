Photobooth

by Ahmad Khairul Anwar

Figoera

Web applications designed to streamline photo booth sessions with integrated Midtrans payment gateways. I led the entire front-end development lifecycle—from architecture and UI/UX design to API integration and deployment—for both projects.

App includes an intuitive admin dashboard for managing photo booth transactions, printable photos, custom frames and filters, discount automation, and QR code-based customer feedback.

Platforms were built with scalability in mind and optimized to handle dynamic photo slideshows, real-time updates, and responsive layouts across devices.
