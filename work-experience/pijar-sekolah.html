As a Middle Front End Developer besides the tasks explained for a junior developer, my other responsibilities includes

  1. Deploying features to release for QA testing and deploying features to production.
  2. Fix errors detected in Jenkins/SonarQube and hotfix errors that occur in production.
  3. Regularly check for outdated technologies in the base code and upgrade them.
  4. Add environments to the vault.
  5. Review and merge code from other front end colleagues.
  6. Periodically supervise and assist new front end colleagues or interns to ensure they align quickly with the company's requirements.

As a Junior Front End Developer, besides the tasks explained for a basic developer, my other responsibilities includes

  1. Providing assistance by reviewing code, discussing feature completion strategies and code syntax with other front end colleagues
  2. Actively participating in discussions with the UI/UX team, PM, and BE regarding upcoming features
  3. Offering suggestions to make the features more user-friendly and targeted.
  4. Additionally, I participate in meetings with third-party stakeholders when necessary.
